# Saturday Meetup Dashboard

부트캠프 토요일 오프라인 모임을 관리하는 Next.js 대시보드입니다.

## 주요 기능

- 오프라인 스터디 대시보드
  - 날짜별 모임 카드 조회
  - 모임별 멤버/엔젤 현황 시각화
  - 모임 상세에서 참여자 추가/수정/삭제, 모임 정보 수정/삭제
  - 팀 할당 현황(팀 필터, 참여 상태, 할당 모임 뱃지)
- 뒷풀이 관리 대시보드
  - 날짜별 뒷풀이 카드 조회
  - 카드에서 참여자 목록, 정산자, 정산 계좌 표시
  - 요약 지표(모임 수, 총 참여, 미정산 인원, 정산 완료율)
  - 뒷풀이 상세에서 참여자 관리 및 팀 기반 빠른 추가
- 정산 관리
  - 참여자별 정산 상태 토글(정산 완료/미정산)
  - 정산자/계좌번호 등록 및 수정
- 멤버 관리
  - 팀/엔젤/멤버 구성 편집
  - 팀 추가/삭제 및 자동 저장
- 도메인 연동
  - 오프라인 스터디와 뒷풀이 간 팀 할당 현황 공유
  - 모임/뒷풀이 전환 시 날짜 컨텍스트 유지
